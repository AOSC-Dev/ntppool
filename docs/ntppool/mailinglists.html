NTP Pool mailing lists

Currently, there are three mailing lists for the pool.ntp.org project.

Announcements

Very very low traffic announcements of NTP Pool news. All server operators should be subscribed to the announcement list.
timekeepers-announce | subscribe | archive

Discussion

Server operators (and users) who wish to be more involved in the project can subscribe to the discussion list.
Of course, general discussions about ntp should take place on the comp.protocols.time.ntp Usenet newsgroup (also on Google Groups).
timekeepers | subscribe | archive

Development

More technical (and ideally always pragmatic) discussions about the current development of the ntp pool code and pgeodns happens on the timekeepers-dev list.
timekeepers-dev | subscribe | archive
